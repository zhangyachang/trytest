## 字符串方法

#### 1. length

```js
.length 可以返回字符串的长度，只是一个只读属性，不可写
```



#### 2. charAt

> [ ] 取对应序列号的字符， [ ]有兼容问题，不兼容ie7及其以下。
>
> a.charAt(0) 可以解决上面的兼容问题。这两个都只是 只读属性。

```js
var str = 'abcded';
str.charAt(3); // d
```



#### 3. charCodeAt

str.charCodeAt() 接收一个字符，返回该字符在 Unicode 码的序号。

```js
var str = 'abcded';
str.charCodeAt(); // 97 默认是第0个
str.charCodeAt(2); // 99
```
